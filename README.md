```js
// 项目介绍
    仿照京东淘宝等做的一个手机端的电商demo，功能包括了登录登出、商品列表、搜索、商品分类、购物车、我的信息、收货地址、我的订单等功能。
```

```js
// 1.依赖包加载
    yarn
// 2.启动项目
    yarn serve
// 3.项目打包
    yarn build
```

```js
// vue-cli 脚手架搭建
    yarn add @vue/cli
    vue create vue-admin
    cd vue-admin
    yarn serve
```

```js
// 技术选型
```
